Sajilo Notes

Sajilo Notes is an online notes portal through which we try to make the quality National Education Board [NEB] notes available to all the students studying in +2 Level in Nepal. The notes are strictly based on the syllabus as prepared by NEB.

BABBLER Prints!

BABBLER Prints is an just-for-fun name manipulation command line open-source application developed using the QBASIC programming language. The code of BABBLER is open-sourced at GitHub It's development is still under progress and we're working to develop a GUI version of this app with additional features.
